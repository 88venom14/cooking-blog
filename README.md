ПРОЕКТ Кулинарный блог(веб)

ПЛАН ПРОЕКТА:

1. Реализовать авторизация/регистрация/гость.

2. Реализовать возможность оставлять комментарии(возможность оставлять комментарии анонимно).

4. Реализовать возможность добавлять ссылки на сторонние соц-сети в профиле пользователя.

5. Реализовать возможность ставить реакции на рецепты.

6. Реализовать возможность создавать рецепты.

7. Реализовать возможность создания групп где можно обмениваться рецептами вместе с другими пользователями

8. Рализовать список "Популярные рецепты", "Лучшие рецепты за месяц", "Подборка простых рецептов", "Подборка сложных рецептов".

9. Реализовать категории рецептов.

Проект выполнялся с помощью Java script с использованием Node.js Базой данных послужила SQlite с использованием Prisma ORM
